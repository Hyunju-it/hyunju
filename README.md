# 이현주의 기술 블로그

성장하는 백엔드 개발자의 기록

## 블로그 주소

<https://hyunju-it.github.io/hyunju/>

## 기술 스택

- **Backend**: Java, Spring Boot, JPA, REST API
- **Database**: MySQL, Redis
- **Infra / DevOps**: AWS, Docker, Kubernetes, Linux, CI/CD

## 로컬 실행

```bash
bundle install
bundle exec jekyll serve
```

## 테마

[Jekyll Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy/) 테마를 사용합니다.

## Usage

Check out the [theme's docs](https://github.com/cotes2020/jekyll-theme-chirpy/wiki).
